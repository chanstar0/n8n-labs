### 核心组件架构
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   定时触发器     │ -> │  AI脚本生成     │ -> │  内容质量评估   │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                                        │
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   执行报告      │ <- │  人工审核点     │ <- │  多媒体处理     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                                        │
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  小红书发布     │    │   抖音发布      │    │   B站发布       │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 技术栈组成
- **工作流引擎**: N8N (企业级配置)
- **AI内容生成**: Rytr API
- **语音合成**: ElevenLabs TTS
- **视频生成**: HeyGen AI Avatar
- **数据库**: PostgreSQL 15
- **缓存**: Redis 7
- **反向代理**: Nginx + SSL
- **监控**: Prometheus + Grafana
- **容器化**: Docker + Docker Compose
